1. 비즈니스 요구사항 정의서 (BRD)

1.1 프로젝트 개요
- 프로젝트명: 리튼(litten) - 크로스플랫폼 노트 앱
- 프로젝트 목표: 다양한 입력 방식(듣기(녹음), 쓰기(키보드 타이핑, 변환된 문서 위에 쓰기))을 지원하는 통합 노트 앱 개발
- 타겟 플랫폼: iOS, Android, Web, Desktop
- 비즈니스 목표: 프리미엄 기능 및 광고 제거를 통한 수익 창출

1.2 비즈니스 가치 제안
- 핵심 가치: 다양한 형태의 노트 작성을 하나의 앱에서 통합 지원
- 차별화 요소: 
  * 3가지 입력 방식 통합 (듣기, 쓰기)
  * 27개 언어 지원으로 글로벌 시장 대응
  * 크로스플랫폼 동기화
  * 직관적인 사용자 인터페이스

1.3 사업 목표
- 단기 목표 (6개월): 50만 다운로드, 5% 프리미엄 전환율
- 중기 목표 (1년): 200만 다운로드, 10% 프리미엄 전환율
- 장기 목표 (2년): 500만 다운로드, 교육 기관 B2B 진출

1.4 성공 지표 (KPI)
- 다운로드 수
- 일일 활성 사용자 (DAU)
- 월간 활성 사용자 (MAU)
- 프리미엄 전환율
- 광고 수익 (eCPM)
- 사용자 만족도 (앱스토어 평점)

2. 제품 요구사항 명세서 (PRD)

2.1 제품 개요
- 제품명: 리튼(litten)
- 제품 유형: 크로스플랫폼 노트 앱
- 타겟 사용자: 학생, 직장인, 연구원, 창작자
- 지원 플랫폼: iOS, Android, Web, Desktop (Windows, macOS, Linux)

2.2 핵심 기능 요구사항

2.2.1 홈 화면
- 리튼은 디렉토리 개념으로 해당 리튼 안에 듣기, 쓰기한 데이터가 있음
- 생성된 리튼 또는 듣기, 쓰기 기능의 파일 리스트가 시간 순서대로 정렬됨 
- 파일을 선택하면 해당 기능의 탭으로 넘어 가며 해당 내용을 플레이, 텍스트 쓰기나 변환된 문서위에 쓰기가 가능
- 리튼을 선택하면 선택된 리튼의 듣기, 쓰기 데이터만 각 탭에서 보기가 가능
- 무료인 경우 광고가 표시
- 유료인 경우 로그인이 필요하며 광고가 제거되고 핸드폰에 있는 내용이 서버와 동기화

2.2.2 듣기 기능
- 음성 파일 듣기, 저장, 재생 
- 음성 파일 저장 및 재생 (1배속, 1.2배속, 1.5배속, 2.0배속 )
- 무료인 경우 광고가 표시
- 유료인 경우 서버에 있는 파일과 동기화

2.2.3 쓰기 기능
- 리치 텍스트 에디터 (볼드, 이탤릭, 하이라이트)
- 마크다운 지원
- 체크리스트 및 태그 기능
- 듣기 중 텍스트 작성이나 필기/스케치 시 해당 내용이 듣기 어느 위치에 해당하는지 자동으로 저장하고 표시
- 텍스트 파일의 제목은 첫글자부터 시작해서 5글자를 제목으로 생성
- 무료인 경우 광고가 표시된다. 
- 유료인 경우 서버에 있는 파일과 동기화 된다. 
- 파일 추가 버튼 클릭 시 폰 내에 있는 파일(PDF,TXT,DOC,PPT)를 선택시 이미지 파일(JPEG)로 변환하여 리튼 폴더 또는 관리 디렉토리로 저장 
- 변환되어 저장되는 파일 종류: JPEG
- 하이라이트, 밑줄, 취소선
- 텍스트 주석 및 스티커 노트
- 드로잉 및 도형 그리기
- 페이지별 노트 관리
- 무료인 경우 광고가 표시
- 유료인 경우 서버에 있는 파일과 동기화
 
2.2.4 설정 기능
- 최대 듣기 시간 : 30분, 1시간, 2시간, 3시간을 선택할 수 있고 기본값은 1시간
- 자동 저장 간격 : 10초, 30초, 1분, 3분, 5분, 10분을 듣기, 쓰기 선택된 시간에 따라 자동으로 저장 기본값은 1분
- 테마: 나라별, 언어별 선호하는 5가지 테마 
- 언어: 27개 언어 선택
- 유무료 상태 및 유료 전환 시 결제로 연동 

2.3 비기능적 요구사항
- **시스템 언어 자동 감지**: 앱이 처음 설치될 때 해당 시스템의 언어를 감지해서 30개 지원 언어 중 해당 언어로 자동 설정 (지원하지 않는 언어는 영어로 기본 설정)
- **국가별 선호 테마 자동 선택**: 앱이 처음 설치될 때 감지된 언어/국가에 따라 해당 지역이 선호하는 5가지 테마 중 자동으로 선택되어 적용
  * 아시아권(한국, 일본, 중국): Classic Blue
  * 유럽권(독일, 프랑스, 이탈리아): Dark Mode  
  * 아메리카권(영어, 스페인어, 포르투갈어): Nature Green
  * 중동/아프리카권(아랍어, 페르시아어, 하우사어): Sunset Orange
  * 기타 지역: Monochrome Grey
- 앱이 처음 설치될 때 UUID를 생성해서 서버로 전송하고 유료로 전환하기 전까지 아이디로 활용
- 상위에는 항상 선택된 리튼명이나, 현재 듣기이나, 쓰기중인 파일명이 표시
- 성능: 앱 시작 시간 3초 이내, 파일 로딩 5초 이내
- 보안: 앱별 아이디, 데이터 암호화, 개인정보 보호
- 접근성: 스크린 리더 지원, 고대비 모드
- 호환성: iOS 12+, Android 7+, 최신 브라우저
- 무료: 로그인 없이 사용 가능
- 무료인 경우 광고가 표시
- 유료인 경우 서버에 있는 파일과 동기화
- 유료: 유료인 경우 PC에서도 로그인하여 관리가 가능

3. 타겟 사용자 분석 및 페르소나

- 특징: 강의 노트, 연구 자료 정리, 논문 작성
- 니즈: 음성 듣기, PDF 주석, 다양한 자료 통합 관리

3.1.2 직장인 (사무직, 관리직)
- 연령: 25-45세
- 특징: 회의 메모, 업무 문서 관리, 프레젠테이션 준비
- 니즈: 빠른 메모, 문서 협업, 크로스플랫폼 동기화

3.1.3 연구원 (학술 연구자, 기업 연구원)
- 연령: 28-50세
- 특징: 논문 분석, 연구 노트, 데이터 정리
- 니즈: 체계적인 자료 분류, 인용 관리, 고급 검색

3.1.4 창작자 (작가, 블로거, 콘텐츠 제작자)
- 연령: 20-40세
- 특징: 아이디어 수집, 초안 작성, 멀티미디어 콘텐츠
- 니즈: 영감 기록, 이미지 편집, 다양한 형식 지원

4. 경쟁사 분석

4.1 직접 경쟁사

4.1.1 Notion
- 강점: 
  * 강력한 데이터베이스 기능
  * 팀 협업 기능
  * 다양한 블록 타입 지원
- 약점:
  * 복잡한 사용법으로 진입장벽 높음
  * 음성 듣기 기능 부족
  * PDF 주석 기능 제한적
- 시장점유율: 노트 앱 시장 15%
- 수익모델: 구독 서비스 (개인 $8/월, 팀 $10/월)

4.1.2 Evernote
- 강점:
  * 오래된 브랜드 신뢰도
  * 웹 클리핑 기능
  * OCR 기능 (유료)
- 약점:
  * 느린 성능
  * 구식 UI/UX
  * 제한적인 무료 기능
- 시장점유율: 노트 앱 시장 12%
- 수익모델: 프리미엄 구독 ($7.99/월)

4.1.3 GoodNotes
- 강점:
  * 뛰어난 PDF 주석 기능
  * 자연스러운 필기 경험
  * 타블렛 최적화
- 약점:
  * iOS/iPadOS 중심 (Android 제한적)
  * 음성 기능 부족
  * 텍스트 편집 기능 약함
- 시장점유율: 디지털 노트 앱 시장 20%
- 수익모델: 일회성 구매 ($7.99)

4.1.4 Microsoft OneNote
- 강점:
  * Office 365 연동
  * 무료 사용 가능
  * 크로스플랫폼 지원
- 약점:
  * 복잡한 인터페이스
  * 느린 동기화
  * 제한적인 PDF 기능
- 시장점유율: 노트 앱 시장 18%
- 수익모델: Office 365 구독 포함

4.2 간접 경쟁사

4.2.1 Google Keep
- 강점: 간단한 사용법, 구글 서비스 연동
- 약점: 기능 제한적, 고급 편집 기능 부족

4.2.2 Apple Notes
- 강점: iOS 기본 앱, 간편한 사용
